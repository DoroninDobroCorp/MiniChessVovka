# 🚀 Быстрая установка автоматического обучения

## Что это делает?

Система автоматически запускает обучение AI каждый день:
- ⏰ **Время работы**: с 2:00 до 10:00 (8 часов каждую ночь)
- 🖥️ **CPU**: использует только 7 из 8 ядер (одно ядро остается свободным)
- 📝 **Логи**: все ходы записываются с временными метками
- 📊 **Прогресс**: можно в любой момент посмотреть, сколько уже обучено

## Установка (3 команды)

```bash
cd /srv/MiniChessVovka
chmod +x setup_auto_training.sh
./setup_auto_training.sh
```

**Готово!** Обучение будет автоматически запускаться каждую ночь в 2:00.

## Проверка прогресса

### Быстрый просмотр

```bash
./view_progress.sh
```

Покажет:
- Статус обучения (запущено/остановлено)
- Сколько игр сыграно
- Размер кэша (количество изученных позиций)
- Статистика побед
- Последние события

### Следить за обучением в реальном времени

```bash
tail -f training_log.txt
```

Вы увидите каждый ход с:
- Временем расчета
- Выбранным ходом
- Оценкой позиции

## Управление

### Запустить сейчас (не дожидаясь 2:00)

```bash
./run_scheduled_training.sh start
```

### Остановить обучение

```bash
./run_scheduled_training.sh stop
```

### Проверить статус

```bash
./run_scheduled_training.sh status
```

## Файлы

После установки появятся:

| Файл | Описание |
|------|----------|
| `training_log.txt` | Подробный лог всех ходов |
| `training_progress.txt` | Сводка прогресса обучения |
| `scheduler.log` | Лог работы планировщика |
| `move_cache.db` | База изученных позиций |

## Оценка времени обучения

При глубине поиска `depth = 6`:

- **Одна партия**: ~3-6 часов
- **За ночь (8 часов)**: ~1-2 партии
- **За месяц**: ~30-60 партий
- **За 3 месяца**: ~90-180 партий

Для **полного обучения** (AI станет очень сильным):
- Рекомендуется: **100-200 партий**
- Это займет: **2-4 месяца** ночного обучения

## Изменение настроек

### Изменить время работы

Отредактируйте `run_scheduled_training.sh`:

```bash
START_HOUR=2   # Начало (2:00)
END_HOUR=10    # Конец (10:00)
```

Например, для работы с 1:00 до 11:00:
```bash
START_HOUR=1
END_HOUR=11
```

### Изменить количество ядер

Отредактируйте `run_scheduled_training.sh`:

```bash
CPU_MASK="0-6"  # 7 ядер (рекомендуется)
# CPU_MASK="0-7"  # Все 8 ядер (максимальная скорость)
# CPU_MASK="0-3"  # 4 ядра (если компьютер нужен ночью)
```

### Ускорить обучение

Отредактируйте `scheduled_self_play.py`, функцию `main()`:

```python
depth = 4  # Вместо 6 (быстрее, но слабее)
```

Или используйте все 8 ядер:
```bash
# В run_scheduled_training.sh
CPU_MASK="0-7"
```

## Удаление

```bash
# Отключить автозапуск
crontab -l | grep -v run_scheduled_training.sh | crontab -

# Остановить обучение
./run_scheduled_training.sh stop
```
